# 🎬 MovieDB

A simple JavaScript movie explorer that fetches and displays movie data in a clean UI.

## 🚀 Features

- Search for movies by title
- Dynamic movie cards with poster, rating, and overview
- Responsive layout with clean styling
- Built with plain HTML, CSS, and JavaScript

## 🧩 Project Structure

- `index.html` – main page markup
- `style.css` – visual styling and layout
- `app.js` – JavaScript logic for movie fetching and search
